# User Management API

## Features
- CRUD operations (GET, POST, PUT, DELETE)
- Input validation
- Logging middleware

## Copilot Usage
Used GitHub Copilot to:
- Debug API errors
- Improve validation logic
- Optimize routes

## Setup

```bash
npm install
node app.js
```

## Endpoints
- GET /users
- GET /users/:id
- POST /users
- PUT /users/:id
- DELETE /users/:id
